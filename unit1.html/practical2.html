Welcome to My Basic HTML Page

About This Page

This is a simple webpage created using onlyHTML. It demonstrates the use of headings, paragraphs, and plain text without any CSS styling.

Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create webpages. Headings are defined with <h1> to <h6> tags, and paragraphs are defined with <p> tags.

Conclusion

This page is a basic example showing how text content can be structured using headings and paragraphs. You can expand it further by adding lists, images, and links.
